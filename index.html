C#

C# (C-Sharp) is a programming language developed by Microsoft that runs on the .NET Framework. C# is used to develop web apps, desktop apps, mobile apps, games and much more.

Java

Java is a popular programming language. Java is used to develop mobile apps, web apps, desktop apps, games and much more. It is known for its flexibility between different platforms and operating systems.

Python

Python is a popular programming language. Python can be used on a server to create web applications. It can be used to control robotics, and automation. It can also be used to make desktop applications and is postentially the most common language for machine learning.

Ruby

Ruby is a general-purpose, interpreted programming language. Ruby is a true object-oriented programming language. Ruby is a server-side scripting language similar to Python and PERL. Ruby can be used to write Common Gateway Interface (CGI) scripts.

PHP

PHP is a server scripting language, and a powerful tool for making dynamic and interactive Web pages. PHP is a widely-used, free, and efficient alternative to competitors such as Microsoft's ASP. It is the most common back end programming language on the Web. Wordpress was built on PHP.

C++

C++ is a popular programming language. C++ is used to create computer programs. It is incredibly powerful, things like Spotify, Youtube, Amazon and Windows were programmed with C++.

JavaScript

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web. JavaScript is easy to learn and a great starting point for any aspiring programmer.

Swift

Swift is a new programming language from Apple, built by Apple, that builds on the techniques learned in Objective-C, but modernizes them to include shorter syntax and easier readability, maintainability and safety to prevent crashes.
